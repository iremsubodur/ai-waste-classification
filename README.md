# ♻️ AI-Based Waste Classification with Grad-CAM

This project classifies waste into glass, metal, paper, and plastic
using a deep learning model with Grad-CAM visual explanations.

## Features
- Image upload
- Live webcam
- Grad-CAM heatmap
- Adjustable intensity
- English / Turkish support

## Run
pip install -r requirements.txt
streamlit run app.py
